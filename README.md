## Commande utilisées

   - Partie commande

        ```bash
        - npx create-react-app app-name --template typescript
        - npm install react-router-dom
        - npm i -D node-sass
        - npm i -D @types/node-sass
        ```

   - Partie navigateur
        
        ```bash
        - npm i
        - npm start
        ```
